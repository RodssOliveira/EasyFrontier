Easy Frontier API

Easy Frontier API para consulta de medicamentos

Estrutura das requisições

Url

É necessário fazer uma chamada HTTP, método GET, para a URL https://www.easyfrontier.com/api/

Parâmetros de URL
Remedios: Para retornar todos os medicamentos registrados no sistema
Remedios/nome_remedio: Para retornar um medicamento especifico
Remedios/nome_remedio/dado_especifico: Para retornar um dado especifico do medicamento

As Respostas são retornadas no formato JSON.

Exemplo de URL
https://www.easyfrontier.com/api/Remedio
https://www.easyfrontier.com/api/Remedio/Tylenol
https://www.easyfrontier.com/api/Remedio/Tylenol/Valor
